# Pick-a-Print Browser Extension

Chrome extension (Manifest V3) to save models from supported sites.

## Install

1. Run the backend (`docker compose up` or `runserver`)
2. Log in → **Settings** → copy API Base URL and Token
3. Open `chrome://extensions` → Developer mode → **Load unpacked**
4. Select this `extension/` folder
5. Click extension icon → paste URL + token → Save

## Supported sites

- Printables
- MakerWorld
- Thangs

A floating **Save to Library** button appears on model pages.
